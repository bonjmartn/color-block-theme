COLOR BLOCK THEME
------------------------------

Thanks for using the Color Block theme. This is a fun, bold, and colorful theme that you can use for any type of WordPress website.

The theme's main informational page is at:
https://www.zenwebthemes.com/shop-themes/color-block/

------------------------------
FREE AND PREMIUM VERSIONS
------------------------------

This theme comes in two versions: Free and Premium.

Free Features

- WordPress Customizer
- Responsive design
- Two social media widgets
- An easy homepage builder that uses the Customizer and widgets
- Call-to-action button and headline widgets
- Large images across the top of pages and posts

Premium Features

The Premium version has all of the free features, plus:

- Customize the background colors of the "color blocks" on the homepage
- Google Fonts
- Typography choices such as colors and sizes
- Full width, left sidebar, and right sidebar Page layouts
- Different sidebars for Posts vs Pages
- Choose a large photo for the top of Archives pages (archives, blog index, categories, tags)
- Get all 20 of the beautiful photos that you see in the theme demo
- Tech support

IMAGE SIZES
-------------------------------------------------------

Recommended Image Sizes

The large photo on the top of the homepage: 1200 px width
The side-by-side photos on the homepage: 600 x 400 px
Featured images for blog posts and pages, which appear at the top of the page: 1000 x 667 px
